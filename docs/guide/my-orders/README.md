# Мои заказы

[https://elements.01.ht/my-orders](https://elements.01.ht/my-orders)

Раздел представляет из себя таблицу с информацией о заказах.

Содержание таблицы:

* Номер заказа
* Статус
* Тип
* Дата
* Сумма
* Детали заказа
* Документы

Также через данный раздел вы можете перейти на страницы оплаты заказа.
